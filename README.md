# CacaoAPP

Aplicación Android para clasificación de cacao usando PyTorch Mobile y Jetpack Compose.

## Características
- Clasificación de imágenes
- Modelos PyTorch (.torchscript)
- Interfaz moderna con Compose

## Requisitos
- Android 8.0 (API 26) o superior
- Android Studio
